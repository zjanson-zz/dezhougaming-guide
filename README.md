# dezhougaming-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不依赖特定域名或后端服务，适合直接通过浏览器访问或部署到各类静态托管平台。

## 项目简介

这是一个资料性的归档仓库，主要存放与德州游戏相关的说明、指南及工具页面。每个 HTML 文件均为独立页面，内容完整，无需额外依赖。仓库本身不指向任何具体网站或服务，仅作为静态页面的集合与发布源。

## 目录说明

```
dezhougaming-guide/
├── index.html          # 仓库首页 / 导航页
├── guide/              # 指南类页面文件夹
├── tools/              # 工具类页面文件夹
└── assets/             # 公共资源（样式、脚本、图片等）
```

- `index.html`：仓库的入口页面，可在此处浏览所有归档页面的索引。
- `guide/`：存放各类指南、说明性 HTML 页面。
- `tools/`：存放辅助工具或计算类 HTML 页面。
- `assets/`：存放页面共用的 CSS、JavaScript 及图片资源。

## 页面归档说明

- 所有页面均为纯静态 HTML，不包含跟踪代码、广告或外部请求（除公共 CDN 资源外）。
- 每个页面独立完整，可直接在浏览器中打开使用。
- 页面内容聚焦于德州游戏的规则、策略、术语解释或辅助工具，不涉及具体平台或推广信息。
- 归档页面会随维护逐步更新或增删，保持内容准确与清晰。

## 维护说明

- 本仓库由维护者不定期更新，添加新页面或修正已有内容。
- 欢迎通过 Issues 提交建议或报告问题（如页面显示异常、内容错误等）。
- 不接受涉及商业推广、具体网址或诱导点击的内容提交。
- 所有页面遵循简洁、中立、实用的原则。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或对应文件夹中的 HTML 文件即可浏览。
3. 也可将整个仓库部署到任意静态托管服务（如 GitHub Pages、Netlify 等）进行在线访问。
